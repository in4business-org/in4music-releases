# in4music — descargas

Builds de in4music para Windows (app de practica de canto y guitarra).

## Como instalar

1. Entra a [Releases](../../releases) y descarga el ZIP del ultimo release
   (`in4music-windows-vX.Y.Z.zip`).
2. Descomprimilo en una carpeta y ejecuta `in4music.exe`.
3. Para actualizar: cerra la app, descomprimi el ZIP nuevo encima de la misma
   carpeta y volve a abrir.

> **Importante:** descarga solo el ZIP. Los links "Source code (zip / tar.gz)"
> los agrega GitHub automaticamente a todos los releases y no se pueden
> ocultar; no contienen la aplicacion (solo este README). Ignoralos.
